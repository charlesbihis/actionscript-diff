public function diff(before:String, after:String):Array

Finds the difference between two text strings. Given the two text strings, will return an array of 'Operation' objects which will describe an efficient (no guaranteed optimal) path from original string to resulting string.

Parameters

before:String — The 'before' string to be diffed.
 
after:String — The 'after' string to be diffed.

Returns
Array — An array of Operation objects describing the difference between the two given strings.
